title: Versions
description: Supported PostgreSQL Versions
---

SchemaHero works with various versions of PostgreSQL.

| PostgreSQL Version | Minimum SchemaHero Version |
|------------------|------------|
| 13-beta | 0.1.0 / v1alpha1 |
| 12.x | 0.1.0 / v1alpha1 |
| 11.x | 0.1.0 / v1alpha1 |
| 10.x | 0.1.0 / v1alpha1 |
| 9.6 | 0.1.0 / v1alpha1 |
| 9.5 | 0.1.0 / v1alpha1 |
